# Earth Rotation Loop

This Remotion source creates the short Earth self-rotation loop used in the
`자전과 공전` learning dialog.

Remotion is used only as an offline production tool. The Vite app loads the
rendered files from `public/learning/` with a normal `<video>` element.

## Commands

```bash
npm run remotion:earth:studio
npm run remotion:earth:still
npm run remotion:earth:render
```

Generated runtime assets:

- `public/learning/earth-rotation.mp4`
- `public/learning/earth-rotation-poster.png`
